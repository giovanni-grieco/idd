# Indexer

In questa cartella ci sono tutti gli strumenti per scaricare i dati, elaborarli e inserirli su elastic search.

# Step da seguire
- Fetch
- Create Index
- Index
- Extract
- Delete Index (quando non serve più).

# Moduli disponibili
- fetch 
- create_index
- index
- extract
- delete_index

# Come lanciare gli script?

Su CLI scrivere:

```bash
python3 -m <nome_modulo>
```

ad esempio
```bash
python3 -m fetch
```
